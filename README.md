# weboodi-chart

## How to develop

1. Clone the repo
2. Run `yarn install` to install development dependencies
3. Run `yarn test` to run tests
4. Run `yarn build` to build the extension
5. Open `chrome://extensions` and click "Load unpacked" to install the dev extension
